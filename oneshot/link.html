AGENT 연동서비스

고객사의 서버에 에스엠티엔티의 SMS/LMS/MMS/카카오 모듈을 설치하여 문자연동 시스템을 구축하는 방식입니다.

  • 서버간 통신(TCP/IP) 기반의 검증된 안전한 통신

  • 업체에 맞는 OS/DB 환경을 지원하여 업체 시스템에 맞도록 적용이 가능

  • 기존 타사 AGENT 연동 모듈 형식지원 (커스터마이징 가능)

API 연동서비스

웹서비스 기반으로 HTTP 요청을 보낼 수 있는 환경이면 문자 연동이 가능합니다.

윈도우, 리눅스, 유닉스, Mac OS 등 모든 OS 환경 지원

구분 AGENT 연동 서비스 API 연동 서비스
운영체제(OS) 환경 LINUX, UNIX(Solaris), Windows, NT 2000 ~ 2012 모든 OS 지원
데이터베이스(DB) 환경 MySQL, MSSQL, Oracle 지원 (기타 DB는 문의) 모든 OS 지원
